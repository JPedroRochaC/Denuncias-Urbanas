# 📢 Denúncias Urbanas

Sistema web para registro e acompanhamento de problemas urbanos, como buracos, lixo irregular e falhas na iluminação pública.

## 🚀 Sobre o projeto

O **Denúncias Urbanas** permite que usuários registrem ocorrências na cidade de forma simples e acompanhem o status de cada denúncia.

A aplicação é composta por:
- Interface web (front-end)
- API em Python (back-end)
- Banco de dados para armazenamento das denúncias

---

## 🛠️ Tecnologias utilizadas

### Front-end
- HTML  
- CSS  
- JavaScript  

### Back-end
- Python  
- Flask  
- Flask-CORS  
- Gunicorn  

### Banco de dados
- MySQL  
- PyMySQL  

---

## 🌐 Deploy

- Front-end: Netlify  
- Back-end: Render / Railway  

---

## ⚙️ Funcionalidades

- Registrar denúncias  
- Listar denúncias  
- Filtrar por tipo  
- Buscar por endereço ou descrição  
- Atualizar estatísticas  
- Excluir denúncias
